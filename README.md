# VoxelisEditor
An app that allows you to make a 3D scene or a game and export it into Catweb.

**This editor allows you to:**
- Build your scene
- Add your own textures
- Export your game into CatWeb

Thanks to __Voxelis__ (by Jexx) for giving me the only reason to make this project.
