# VoxelisEditor
Want to make your own 3D game in Catweb? This tool is made for you.

## Note that it is WIP, most stuff don't even work.

**This editor allows you to:**
- Build your scene
- Add your own textures
- Export your game into CatWeb

Thanks to __Voxelis__ (by Jexx), we have 3D in Catweb. This tool just allows you to easily make the 3D Games.
